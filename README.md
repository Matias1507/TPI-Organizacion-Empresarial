# Bot de Gestión de Pedidos - El Imperio

Este repositorio contiene la simulación del sistema automatizado de reabastecimiento para el restaurante "El Imperio", desarrollado como Trabajo Práctico Integrador para la cátedra de Organización Empresarial (UTN).

## Cómo ejecutar el proyecto

1. **Requisitos:**
   - Tener instalado [Python 3.x](https://www.python.org/).

2. **Pasos:**
   - Descargar o clonar este repositorio.
   - Abrir la consola en la carpeta del proyecto.
   - Ejecutar el programa con el siguiente comando:
```bash
     python main.py
     ```

## Características técnicas
- **Persistencia:** Simulación de base de datos relacional mediante estructuras de datos en memoria.
- **Robustez:** Implementación de manejo de errores (Camino Infeliz) mediante bloques `try/except`.
- **Arquitectura:** Basada en una Máquina de Estados Finita (FSM) para garantizar un flujo lógico y ordenado de la información.
